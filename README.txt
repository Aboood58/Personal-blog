ملف المشروع: مدونة شخصية جاهزة للرفع (Static website)

الملفات:
- index.html  -> الملف الرئيسي (RTL, Arabic)
- styles.css  -> تنسيقات CSS
- script.js   -> سكربت بسيط لحفظ المحتوى محلياً (localStorage) وإمكانية إضافة محتوى من الواجهة
- assets/     -> يمكنك إضافة صور أو ملفات هنا قبل الضغط والرفع

كيف أرفع الموقع إلى استضافة (بشكل عام):
1) يمكنك رفع الملفات كما هي إلى أي استضافة تقدم ملفات ثابتة (Static Hosting) مثل GitHub Pages, Netlify, Firebase Hosting أو استضافة تقليدية.
2) لضغط المشروع: استخدم الأرشيف (zip) المرفق في هذا الحزمة.
3) بعد الرفع، سيكون الموقع متاح مباشرة عبر الرابط الذي توفره خدمة الاستضافة.

تلاحظ:
- الموقع يعمل تماماً بدون خادم (pure static).
- لتضمين فيديو يوتيوب، ضع رابط iframe أو رابط المشاركة في خانة "المحتوى" عند اختيار نوع "video".
- لتغيير النص الترحيبي أو الألوان، عدل index.html و styles.css مباشرة.

تم الإنشاء تلقائياً من قِبل مساعد التصميم — إذا تريد نسخة React/Next.js أو لوحة تحكم أكثر تطوراً أخبرني.
